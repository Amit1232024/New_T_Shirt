# Azf-T Shirt E-Commerce Experience

A polished, single-page e-commerce experience for Azf-T Shirt, showcasing premium T-shirt collections with immersive art direction, experiential storytelling, and a lightweight cart demo.

## Getting started

1. Open `index.html` in your preferred browser.
2. Explore the interactive filters, cart modal, and responsive layout.

## Project structure

```
New_T_Shirt/
├── index.html
├── assets/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── app.js
└── README.md
```

## Features

- Gradient-rich hero with floating studio callout, quick stats, and brand tags
- Experience overview spotlighting virtual fit concierge, material lab, and creator community
- Curated collection highlights, production timeline "Design Lab" section, and seasonal color palette grid
- Dynamic product grid with search, category, and fit filters plus upgraded card styling
- Modal-based search and cart experiences with demo interactions and animated scroll reveals
- Responsive layout optimized for desktop, tablet, and mobile with horizontal lookbook slider controls
- Sustainability, press, story, and testimonial sections to build trust

## Customization tips

- Update the `products` array in `assets/js/app.js` to add or modify available T-shirts.
- Replace Unsplash image URLs with your own product photography.
- Adjust brand colors and typography by editing CSS variables at the top of `assets/css/styles.css`.
- Hook the cart modal into a backend or e-commerce platform by replacing the placeholder logic in `app.js`.

Enjoy building with Azf-T Shirt!
